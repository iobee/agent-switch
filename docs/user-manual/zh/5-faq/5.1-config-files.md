# 5.1 配置文件说明

CC Switch 默认将数据保存在本机。主要数据包括供应商配置、Gateway 状态、用量日志和设备设置。

## CC Switch 数据目录

默认位置：

```text
~/.cc-switch/
├── cc-switch.db      # SQLite 数据库
├── settings.json     # 本机设备设置
├── config.json       # 旧版供应商配置兼容文件
└── backups/          # 本地备份文件
```

`cc-switch.db` 保存供应商、端点、故障转移状态、用量日志、价格数据、代理设置和同步元数据。

## 客户端配置目录

切换供应商或启用 Gateway 路由时，CC Switch 会把对应配置写入目标客户端。

| 客户端 | 默认目录 |
|--------|----------|
| Claude Code | `~/.claude/` |
| Claude Desktop | 系统对应的 Claude Desktop 配置目录 |
| Codex | `~/.codex/` |
| Gemini CLI | `~/.gemini/` |
| OpenCode | `~/.config/opencode/` |
| OpenClaw | `~/.openclaw/` |
| Hermes | `~/.hermes/` |

这些目录可以在设置中覆盖。

## 导入、导出与同步

数据导出会生成 SQL 备份，包含：

- 供应商配置
- Gateway 与代理设置
- 故障转移和流式检查状态
- 用量日志和价格数据
- 应用设置

WebDAV 和 S3 同步使用同样的数据库快照格式，并附带 manifest 进行兼容性检查。
